With advances in genomics, transcriptomics, metabolomics and proteomics, and the more expansive electronic clinical record monitoring, as well as other related efforts, we have entered the Big Data era in biomedical research. Big Data does not have to be defined by sheer size, i.e., petabytes of data, but by the fact that almost all the variables of a system can be measured. Data gathering in systems biology and systems pharmacology is growing rapidly while only a small fraction of the data is converted to useful knowledge. Computational tools and databases are also growing rapidly with an attempt to organize and integrate data for making predictions by performing virtual experiments that impute missing values given the already existing data.
